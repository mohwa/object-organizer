toArray

This function convert an argument like an array like object or an iterable object to an array object.

toArray(v: any, f: function, _this: any): Array
Parameters
v (any) Value which will be convert
f (function) Function which will call on every element of an array object
_this (any) Value which will use as this when executing function
Returns
Array:
Example
toArray([1, 2, 3], function(v) { return { v, vv: this }; }, { x: 1 });
toArray({ x: 1, y: 2, z: 3 });

copyWithin

This function return new an array object that copied every elements from start index until end index just like the Array.prototype.copyWithin() method

copyWithin(v: Array/Object, t: number, start: number, end: number): Array
Parameters
v (Array/Object) An array object
t (number) Target index which to copy the sequence to
start (number) Start index which to start copying elements from
end (number = v.length) End index which to end copying elements from(this function would increase 1 an end index)
Returns
Array:
Example
copyWithin([1, '2', 3], 0, 1, 2)
copyWithin({ x: 1, y: '2', z: 3 }, 0, 1, 2)
